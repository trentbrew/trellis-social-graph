# Plan: Generic Trellis Social Graph Starter

## Goal
Transform the blackwords.co-specific quote archive into a generic **Trellis Social Graph Starter** — a clone-and-go Nuxt scaffold where the demo content is a social feed (posts, people, tags) rather than a curated quote collection.

## New Graph Model

```
Person ←author— Post —tags→ Tag[]
```

Replaces the `Person ←author— Quote —tags→ Tag[]` model. The types are renamed and reframed as social primitives:

| Type   | Role                        | Key fields                                        |
|--------|-----------------------------|---------------------------------------------------|
| Person | A user in the social graph  | `name`, `bio`, `avatarUrl`                        |
| Post   | A social post/entry         | `content`, `likes`, `reposts`, `published` (bool) |
| Tag    | Thematic tag                | `label`                                           |

Relations: `Post --author--> Person`, `Post --tags--> Tag[]`
